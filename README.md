# hlsw-fixes
Fixes and adds a few HL2 Mod Games in HLSW
## Adds support for new HL2 Mod Servers in HLSW
* No More Room in Hell
* Double Action: Boogaloo
* Black Mesa: Deathmatch
* Contagion

## Fixes HLSW bugs
* Fixes Rcon not working for NMRIH and others
